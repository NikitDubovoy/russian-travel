# Проектная работа №3 Яндекс.Практикума. Путешествие по России. 

## Описание.

Сайт разработан при помощи HTML и CSS, адаптирован для разных экранов (от 320px и выше).

## Технологии
1. Flexbox 
2. Grid Layout
3. Относительные величины
4. Медиазапросы
